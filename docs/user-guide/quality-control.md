# Quality Control

!!! info "Coming soon"
    This page is currently under construction. The `qc` module provides tools for
    assessing data quality:

    **Quality metrics:**
 
    - [`compute_tsnr`][confusius.qc.compute_tsnr]: Compute temporal signal-to-noise ratio.
    - [`compute_cv`][confusius.qc.compute_cv]: Compute coefficient of variation.
    - [`compute_dvars`][confusius.qc.compute_dvars]: Compute temporal derivative of variance (DVARS).

    Please refer to the [API Reference](../api/qc.md) and [Roadmap](../roadmap.md) for
    more information.
