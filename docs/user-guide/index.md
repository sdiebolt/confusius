# User Guide

Welcome to the ConfUSIus user guide. This section provides comprehensive documentation
on how to use ConfUSIus for fUSI data analysis, walking you through the various features
and functionalities of the package.

## Overview

ConfUSIus aims to fulfill three primary purposes: read and write fUSI data, perform
statistical analysis of derived metrics (beamformed IQ, power Doppler, velocity, etc.),
and create publication-quality figures. To achieve these goals, ConfUSIus provides a
comprehensive toolkit for working with fUSI data, including:

- Loading and saving data in various formats.
- Processing of beamformed IQ signals.
- Denoising of power Doppler data.
- Quality control metrics.
- Image registration and motion correction.
- Statistical analysis.
- Visualization and exploration.
